# geotools
Some geographical tools for python developers

## Introduction
GeoTools is a small Python library for performing geographical computations.

### Requirements
* ``` cpc.geogrids v0.2.3 ```
* ``` geopandas v0.3.0 ```
* ``` networkx v2.1 ```
* ``` numpy v1.14.3 ```
* ``` osgeo v2.2.4 ```
* ``` pyproj v1.9.5 ```
* ``` rasterio v0.36.0 ```
* ``` rtree v0.8.3 ```
* ``` shapely v1.6.4 ```
