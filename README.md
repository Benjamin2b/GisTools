# geotools
Some geographical tools for python developers

## Introduction
GeoTools is a small Python library for performing geographical computations.

### Requirements
* ``` cpc.geogrids 0.2.3 ```
* ``` geopandas 0.3.0 ```
* ``` networkx 2.1 ```
* ``` numpy 1.14.3 ```
* ``` osgeo 2.2.4 ```
* ``` pyproj 1.9.5 ```
* ``` rasterio 0.36.0 ```
* ``` rtree 0.8.3 ```
* ``` shapely 1.6.4 ```
